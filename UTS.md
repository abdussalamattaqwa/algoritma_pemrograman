## 3. Konversi Suhu
Buat program untuk mengonversi suhu dari **Celcius** ke **Fahrenheit**.  
Gunakan rumus:
```
F = (C * 9/5) + 32
```

**Contoh Eksekusi Program:**
```
Input Celcius: 25
Output Fahrenheit: 77
```
